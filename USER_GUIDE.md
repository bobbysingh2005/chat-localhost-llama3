# User Guide: Chat Localhost Llama3

## Getting Started
1. Open the app in your browser (http://localhost:8000 or http://localhost:5173 for dev).
2. Sign in or use "Continue as Guest" to start chatting.
3. Type or use voice mode (microphone button) to interact with the AI.
4. View, copy, or edit code blocks in chat.
5. Manage your conversation history from the sidebar.

## Features
- Secure login/register and guest mode
- Voice mode (STT/TTS) for hands-free chat
- Copy/edit code blocks
- Accessible UI (screen reader, keyboard navigation)
- REST and MCP chat modes

## Tips
- Use the "New Chat" button to start a fresh conversation.
- For best voice experience, use Chrome or Edge.
- Guest conversations are saved locally; sign in to save to your account.

## Troubleshooting
- If you can't log in, check your credentials or register a new account.
- If voice mode doesn't work, check browser permissions and compatibility.
- For technical issues, contact the developer or check the README for setup help.
